## Om projektet
Hämtar ramschema från JSON med FetchAPI (async/await + try/catch) och skriver ut i en tabell.

## Funktioner
- Sortering genom klick på kolumnrubriker (kurskod, kursnamn, progression)
- Live-sökning (filtrerar på kurskod och kursnamn)

## Teknik
- Vite (Vanilla JavaScript)
- Git & GitHub Actions (automatisk publicering till GitHub Pages)
